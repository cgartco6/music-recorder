# AI Music Recorder Pro

High-definition music recorder with:
- Noise cancellation (removes cars, barks, voices)
- Real-time BPM, genre, instrument detection
- Full lyrics extraction
- AI character singer (upload photo, lip-sync, eye blink, emotions)
- Realistic instrument visualizers (drums, guitar, piano, bass, vocals, synth)
- Exact song matching and replication
- Long-term memory (IndexedDB)
- Download recordings + metadata

## Installation

```bash
npm install
npm start
